Team.EXE - Voice Transfer

Cloning a voice requires hours of recorder speech to build a dataset and then using the dataset to build a new model for the next user. But using transfer learning and Real Time TTS(Text to Speech Synthesis) there is no need to waste our precious time more. We are going to make a toolbox in which you can record your voice and this toolbox will convert your voice into numerical embeddings by which voice from a user can be converted to the voice of another user using autoencoder embeddings.
